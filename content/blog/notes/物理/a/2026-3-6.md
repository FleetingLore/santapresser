# 矢量

矢量 $\mathbf A$, 写作 $\overrightarrow A$.

其大小为

$A = \parallel A\parallel _2$

# 直角坐标系

有三个单位矢量 $i$, $j$, $k$, 分别表示沿 $x$, $y$, $z$ 三轴方向的单位矢量

大小:

$$\mid \vec i\mid = \mid \vec j\mid = \mid \vec k\mid = 1$$

方向余弦

$$\cos \alpha = \frac x r$$
...

# 矢量的基本运算

**平行四边形法则**, **三角形法则**.

$$\overrightarrow C = \overrightarrow A + \overrightarrow B$$

一个矢量可以写为他的模乘以与它同向的单位矢量.
