## 齐次方程

### 类型与解法

$$\frac {\text dy}{\text dx} = f(x, y)\to \frac{\text dy}{\text dx} = \varphi \left(\frac y x \right)$$

### 举例

$$x^3y\text dx + x^2y^2\text dy + x^4\text dx = 0$$

$$x^3y + x^2y^2\frac{\text dy}{\text dx} + x^4 = 0$$

此处如果不容易看出来，可以设 $u = \frac x y$

$$\frac{\text dy}{\text dx} = -\frac x y - \left(\frac x y\right)^2 = 0$$

$$u + x\cdot u' = \varphi (u)$$

$$x\cdot \frac{\text du}{\text dx} = \varphi (x) - u$$

$$\frac{\text du}{\varphi (u) - u} = \frac{\text du}{x}$$

$$\int \frac{\text du}{\varphi (u) - u} = \int \frac{\text du}{x}$$

...
